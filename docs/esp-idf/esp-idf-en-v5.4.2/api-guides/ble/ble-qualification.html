Since 1 July 2024, the identifying number for a new qualified design has changed from Qualified Design ID (QDID) to Design Number (DN). Please log in to the Bluetooth SIG website to view Qualified Product Details, such as Design Details, TCRL Version, and ICS Details (passed cases) and etc.

[2] (1,2)

Some features of the Bluetooth Core Specification are optional. Therefore, passing the certification for a specific specification version does not necessarily mean supporting all the features specified in that version. Please refer to Major Feature Support Status for the supported Bluetooth LE features on each chip.
